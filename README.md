# 🤝 화성 AI 시민리더 잇다(IT-DA)

> 화성특례시 AI 혁신학교 AI랩 공식 플랫폼  
> AI 시민 리더 양성과정 통합 관리 웹 서비스

---

## 📌 프로젝트 개요

**잇다(IT-DA)** 는 화성특례시가 운영하는 AI 시민 리더 양성과정의 수강 신청, 진행 현황 관리, 강사 매칭까지 하나의 플랫폼에서 제공하는 웹 서비스입니다.

화성시 거주 시민이 AI 교육을 받고, 지역 사회에 강사로 파견되기까지의 전 과정을 디지털로 관리합니다.

---

## 🎯 주요 기능

### 👤 시민(수강생)
| 기능 | 설명 |
|------|------|
| 수강 신청 | 비회원 상태로 교육 과정 신청 가능 |
| 화성시 거주 인증 | 주민등록등본 등 서류 제출 → 운영자 확인 |
| 참여코드 회원가입 | 선정 후 발급된 코드로 계정 생성 |
| 나의 현황 대시보드 | 7단계 진행 현황, 다음 수업 안내, 체크리스트 |
| 강사 매칭 연결 | 수료 후 강사 파견 매칭 플랫폼 연결 |

### ⚙️ 운영자
| 기능 | 설명 |
|------|------|
| 거주 인증 관리 | 신청자 서류 승인 / 반려 처리 |
| 참여코드 발급 | 선정자 개별 및 일괄 코드 발급 |
| 기수 관리 | 모집 현황 및 기수별 통계 확인 |
| 수강생 현황 | 전체 수강생 진행 상태 및 출석률 관리 |
